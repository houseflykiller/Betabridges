Install Instructions 
Requirements: Apache 2.2+, Tracker Server (tracker server must be installed to use this app), PHP 7.2  
1. Copy/Upload the folder and files to your web server using cPanel or FTP 
2. If you are installing in sub folder, please make sure there is no space in folder names 
3. GO to assets/apps/conf folder and edit conf.php file. 
4. Enter your server URL/IP and Google Map API Key. 
5. Now go to open your URL and login with server credentials.

6. To change logo go to assets/img folder.
7. To change dashboard logo change logo.jpg file with you base64 file.
